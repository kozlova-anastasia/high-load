# Snaply — Социальная сеть для обмена визуальным контентом

## 1. Тема и целевая аудитория

### 1.1 Тема проекта

**Snaply** — это социальная платформа для публикации и потребления фото- и видеоконтента.  
Пользователи могут создавать публикации, подписываться друг на друга, взаимодействовать через лайки, комментарии и личные сообщения.

Проект реализует основные механики Instagram:
- визуальная лента
- профиль пользователя
- подписки
- система рекомендаций
- личные сообщения

---

## 2. MVP-функционал

### 2.1 Публикация контента
- Загрузка фото
- Добавление описания
- Хэштеги
- Геолокация
- Удаление и редактирование публикаций

### 2.2 Главная лента
- Публикации от подписок
- Алгоритмическая выдача
- Infinite scroll

### 2.3 Профиль пользователя
- Аватар
- Биография
- Количество подписчиков
- Количество подписок
- Сетка публикаций

### 2.4 Социальные взаимодействия
- Лайки
- Комментарии
- Подписка / отписка
- Сохранение публикаций

### 2.5 Поиск
- Поиск пользователей
- Поиск по хэштегам
- Поиск публикаций

---

## 3. Ключевые продуктовые решения

- Медиафайлы хранятся в object storage
- Используется CDN для быстрой загрузки изображений
- Лента формируется через систему рекомендаций
- Чат реализован как отдельный сервис
- Кэширование популярных публикаций
- Механизмы модерации и антиспам-фильтрация

---

## 4. Целевая аудитория

- Пользователи 16–35 лет
- Активные потребители визуального контента
- Блогеры и авторы
- Малый бизнес и самозанятые
- Люди, заинтересованные в самопрезентации и социальных связях

География: Россия и страны СНГ.
